Quran Mushaf Reader

A calm, desktop-focused digital Mushaf reader built to make Quran reading in the browser feel closer to reading from a physical Mushaf.

The project focuses on a respectful and distraction-free reading experience, with a polished open-book interface, smooth page navigation, multiple visual themes, bookmarks and notes, Google authentication for syncing personal data, trusted Tafsir integration, and ayah/page audio playback starting with Sheikh Muhammad Siddiq Al-Minshawi.

Main features:
- Digital Mushaf page reader
- Premium open-book visual design
- RTL-aware page navigation
- Page flip interaction
- Multiple reading themes
- Google login with private user data
- Synced bookmarks, notes, and last-read page
- Trusted simplified Tafsir panel
- Ayah and page audio playback
- Local-to-cloud migration for saved marks
- Desktop-first responsive design
